Install Norminette via Bash

1-Go to  https://github.com/42School/norminette

2- install norminette:
>python3 -m pip install --upgrade pip setuptools
>python3 -m pip install norminette

3- permanently add the directory /var/data/python/bin to your PATH environment variable
>nano ~/.bashrc
>export PATH=$PATH:/var/data/python/bin
>source ~/.bashrc
